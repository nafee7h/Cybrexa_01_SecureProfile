About me

I’m a cybersecurity enthusiast and BCA student focused on web application security and penetration testing. I actively work on hands-on labs and projects to understand how real systems behave and where they fail. I’ve performed reconnaissance using tools like Nmap and Amass, discovered hidden endpoints with FFUF and Gobuster, and analyzed HTTP/HTTPS traffic using Burp Suite to understand how web applications process requests and handle data. Through lab environments and CTF-style practice, I’ve identified and tested vulnerabilities such as Cross-Site Scripting (XSS), SQL Injection, and Insecure Direct Object References (IDOR). I’m particularly interested in understanding how these vulnerabilities occur and how they can be prevented in real-world applications. I’ve also built a TCP Packet Analyzer using Python (Scapy) to capture and analyze network traffic, helping me understand TCP/IP behavior and packet-level communication. Currently, I’m focused on strengthening my practical skills, building real-world projects, and continuously improving my understanding of application security.

My Goal

To become a skilled cybersecurity professional specializing in penetration testing by continuously building practical experience through real-world projects, hands-on labs, and vulnerability analysis. I aim to understand how systems are attacked and secured by actively working on web application security, network analysis, and exploiting common vulnerabilities like XSS, SQL Injection, and IDOR, while consistently improving my technical and problem-solving skills.
